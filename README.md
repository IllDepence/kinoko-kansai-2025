# IT + α | IT以外の分野への熱心も生き残るITキャリア

A talk given on 2025-06-26 in Kyoto at 【非公式】きのこカンファレンス in 関西.

Compile with `$ latexmk -lualatex slides.tex`.  
If you're looking at this on GitHub, you should find the compiled slides in the releases.

## Description

学部の頃、「ITが一番楽しい気がするけれど、言語(学)にもすごく興味ある ... どうしよう？」と悩んでいたtarekが歩んできた今までのキャリアをお話します。ITセキュリティを中心に学んだときに言語交流ウェブサイトの脆弱性を報告したエピソードや、機械学習ゼミで指導したロシア系の学生から学んだこと、海外に転職して第二外国語で仕事することになった今。IT一筋とはいえ、IT以外の関心分野も常に楽しめています。ITキャリアを検討している多趣味のあなたも、ぜひITの世界に没入してみてください！

## Target Audience

1. 多趣味の、ITキャリアを検討している人
2. IT以外の分野にも興味があってキャリアの進め方について悩んでいる人
3. 海外でのキャリアを考えている人

## Message

ITキャリアを歩みながら様々なIT以外の分野も楽しめます！
